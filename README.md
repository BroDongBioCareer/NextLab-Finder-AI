# NextLab AI - 博士后职位智能搜索
这是一个基于Google AI开发的博士后职位匹配工具，帮助研究人员快速找到全球学术机会。

作为过来人，知道找博后/RA/博士 位置多耗时！我用Google AI技术做了这个匹配工具：
🔗 直达链接：https://nextlab-ai-1021230570004.us-west1.run.app

✨ 核心亮点：
✅ 语境理解搜索（比如“想多存钱”自动匹配高薪资低消费地区）
✅ 覆盖博士→博士后→研究员全阶段
✅ 按生活方式筛选（户外/城市/家庭友好）
✅ AI自动补全薪资、截止日期等关键信息
✅ 每个职位都可追溯到大学官网，真实可靠

目前还在持续优化中，欢迎各位科研小伙伴试用反馈！
